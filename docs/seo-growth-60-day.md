# 60-Day SEO Execution Runbook

## Weekly Scorecard
- Coverage target: reduce `non-indexed` from `93` to `<40`.
- Impression target: grow from `96` to `600+`.
- CTR target: improve from `3.1%` to `5%+`.
- Primary KPI: `organic_telegram_click` events by landing page.

## 14-Day Metadata Iteration Loop
Run this every 14 days on top pages by impressions:
1. Export GSC `Pages` and `Queries` for last 28 days.
2. Select top 6 pages with high impressions and low CTR.
3. Update title and meta description intent match.
4. Re-check snippet parity in rendered HTML.
5. Compare CTR delta after 14 days and keep winning variants.
